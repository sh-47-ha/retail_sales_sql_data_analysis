# retail_sales_sql_data_analysis

# Project Overview

This project demonstrates the use of SQL to analyze retail sales data and extract meaningful business insights. It covers database creation, data cleaning, exploratory data analysis (EDA), and business-focused SQL queries using a retail sales dataset.

The goal is to practice SQL concepts while solving real-world business problems such as identifying top customers, analyzing sales performance, and understanding customer purchasing behavior.

# Objectives

Create and manage a retail sales database.
Clean and validate the dataset.
Perform exploratory data analysis using SQL.
Solve business problems through SQL queries.
Generate actionable insights from retail sales data.
